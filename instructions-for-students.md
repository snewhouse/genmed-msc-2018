## MAC

- Got to github: 
- download folder

Move in the folder you just downloaded.

```bash
cd genmed-msc-2018
```

change file permissions of the security key

```bash
chmod 600 teaching-gen-med-msc.pem
```

use ssh to connect to AWS

```bash
ssh -i ./teaching-gen-med-msc.pem
```
